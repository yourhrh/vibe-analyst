# Vide Analyst - IDEO-Style Design Thinking

## Phase 1: Empathize - 사용자의 진짜 순간을 관찰하다

### "주식 분석의 감정 곡선" - 실제 사용자가 겪는 감정 여정

```
감정
 😊 ┤                                          "드디어 결론!"
    │                        "오 이건 좋은데?"      ╱
 😐 ┤  "이거 분석해볼까"        ╱              ╱
    │      ╲              ╱              ╱
 😟 ┤       ╲          ╱              ╱
    │        ╲      ╱    "이 숫자가
 😫 ┤         ╲  ╱        맞나...?"
    │          ╲╱
 😤 ┤       "어디서부터
    │        시작하지..."
    └──┬──────┬──────┬──────┬──────┬──────→ 시간
     탐색   수집   분석   검증   결론
     5min  30min  60min  30min  ???

     ▲                              ▲
     │                              │
  우리가 없애야 할               우리가 만들어야 할
  "시작의 막막함"               "확신의 순간"
```

### 핵심 인사이트 3가지

**Insight 1: "분석의 시작이 가장 무섭다"**
> 빈 엑셀, 빈 노트. 어디서부터 시작해야 할지 모르는 공포.
> 사용자는 "도구"를 원하는 게 아니라 "이미 시작된 대화"를 원한다.

**Insight 2: "숫자보다 맥락이 먼저다"**
> PER 15배라는 숫자는 의미 없다. "이 산업에서 15배는 싼 건가 비싼 건가"가 중요하다.
> 사용자는 데이터가 아니라 "판단의 프레임"을 원한다.

**Insight 3: "혼자 결정하는 게 두렵다"**
> 확신이 없으면 행동하지 않는다. 가장 큰 드롭오프는 "분석 완료 후 행동 전".
> 사용자는 "동의해주는 누군가"가 아니라 "다양한 시각의 대화 상대"를 원한다.

## Phase 2: Define - "How Might We..."

```
HMW #1: 어떻게 하면 분석의 "시작 비용"을
        제로에 가깝게 만들 수 있을까?

HMW #2: 어떻게 하면 숫자가 아닌
        "이야기"로 기업을 이해하게 할 수 있을까?

HMW #3: 어떻게 하면 사용자가 "혼자 결정한다"가 아니라
        "여러 전문가와 상의했다"고 느끼게 할 수 있을까?
```

## Phase 3: Ideate - 구체적 디자인 솔루션

### Screen 1: 홈 - "대화의 시작"

기존 금융 앱처럼 대시보드로 시작하지 않는다. 대화로 시작한다.

```

│                                                         │
│                                                         │

│                                                         │

│                                                         │
│     ┌───────────────────────────────────────┐           │
│     │  AAPL, 테슬라, 엔비디아...        🔍  │           │
│     └───────────────────────────────────────┘           │
│                                                         │
│      한글 회사명, 영문 티커, 뭐든 괜찮아요                │
│                                                         │
│                                                         │
│     ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─                │
│                                                         │
│      지금 많이 찾는 기업                                  │
│                                                         │
│     ┌─────────┐ ┌─────────┐ ┌─────────┐               │
```

**디자인 의도:**
- 검색창 하나가 전부. Google처럼 "여기에 치면 된다"가 즉시 명확
- 한글 입력도 OK ("삼성" → 삼성전자 ADR 연결) - 시작의 마찰 최소화
- "지금 많이 찾는 기업" → 뭘 분석할지 모르는 사람에게 시작점 제공
- 배경은 여백 많은 미니멀, 금융앱의 압도적인 숫자 범벅과 차별화

## Phase 4: Design Principles - 디자인 시스템의 감정

### 컬러 시스템

```
Color Philosophy: "차분한 신뢰"

투자 분석은 감정이 아닌 이성의 영역.
자극적인 빨강/초록 대신, 차분하고 권위있는 톤을 쓴다.

Primary
  #1a1a2e  Deep Navy       — 신뢰, 전문성, 깊이

Accent
  #e2b857  Muted Gold      — 가치, 통찰, 프리미엄 (과하지 않게)

Semantic (절제된 사용)
  #4a9079  Sage Green      — 긍정/강점
  #c75a3a  Terracotta      — 부정/리스크
  #8b8fa3  Slate Gray      — 중립

Background
  #fafaf8  Warm White      — 메인 배경
  #f5f3ef  Warm Gray       — 카드 배경

사용하지 않는 것:
  · 주식 앱 특유의 강렬한 빨강/초록 (감정 자극 방지)
  · 네온, 그라디언트 (신뢰도 저하)
  · 순수 흰색 배경 (차가움. Warm White로 대체)
```

### 타이포그래피

```
Typography: "읽히는 것이 아니라, 이해되는 글"

Heading:  Pretendard (한글) / Inter (영문+숫자)
Body:     Pretendard
Mono:     JetBrains Mono (재무 수치)

재무 수치 표기 원칙:
  · 큰 숫자는 항상 단위 축약: $394,000M → $394B
  · 변화율은 항상 동반: $394B (+8.2%)
  · 숫자에는 Mono 폰트 → 자릿수 정렬이 자연스러움
  · 부정적 수치에 빨간색 쓰지 않음 → Terracotta + ▼
```

### 마이크로인터랙션

```
Micro-interactions: "매 순간의 디테일"

1. 밸류에이션 슬라이더
   슬라이더 이동 → 적정가 숫자가 부드럽게 카운트업/다운
   (spring animation, 300ms)
   레인지 바의 현재가 마커가 실시간으로 색상 변화
   (적정가 내 = Gold, 적정가 초과 = Terracotta)

2. 투자 대가 카드 전환
   카드 간 스와이프 시 아바타가 살짝 고개를 끄덕이는
   subtle animation → 인격적 느낌 강화

3. 분석 완료 순간
   프로그레스 100% → 부드러운 confetti 아닌
   "체크마크가 천천히 그려지는" 모션
   → 금융 도구의 무게감 유지

4. 숫자 등장
   재무 수치가 화면에 나타날 때 0에서 카운트업
   (stagger animation, 각 카드 50ms 딜레이)
   → "계산 결과를 보여주는" 느낌

5. 강점/리스크 토글
   강점만 보기 ↔ 리스크만 보기 토글 시
   반대 항목이 blur 처리되며 서서히 사라짐
   → "의도적으로 한쪽만 보고 있다"를 인지시킴
   → 확증 편향 방지의 인터랙션 구현
```

## Phase 5: Information Architecture

```
                            VIDE ANALYST
                                │
                ┌───────────────┼───────────────┐
                │               │               │
             홈/검색          내 분석         설정
                │               │               │
                ▼               ▼               ▼
         ┌──────────┐    워치리스트          프로필
         │ 종목 선택 │    분석 히스토리       알림 설정
         └────┬─────┘    알림 관리           구독 관리
              │
              ▼
        ┌───────────┐
        │ 분석 유형  │
        │ 선택      │
        └─────┬─────┘
              │
     ┌────────┼────────┬────────────┐
     ▼        ▼        ▼            ▼
   종합     대가      밸류        전체
   리포트   분석      에이션      분석
     │        │        │
     ▼        ▼        ▼
  [각 섹션은 독립적으로 접근 가능한 탭]
  [탭 간 자유로운 이동]
  [하단: 다음 행동 유도 (CTA)]
```

**네비게이션 원칙:**
- **깊이는 최대 3단계**: 홈 → 종목 → 분석 탭
- **탭 간 이동은 자유**: 종합 리포트 보다가 밸류에이션 탭으로 바로 이동
- **뒤로가기는 항상 예측 가능**: 탭 내 스크롤 → 탭 선택 → 종목 선택 → 홈
