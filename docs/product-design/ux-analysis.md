# Vide Analyst - UX 분석 & 기능 설계

## 1. 타겟 사용자 페르소나

| 페르소나 | 특징 | 핵심 니즈 |
|---------|------|----------|
| **A. 직장인 투자자** | 시간 부족, 미국 주식 관심, 기본 재무지식 보유 | 빠르고 신뢰할 수 있는 요약 분석 |
| **B. 준전문 개인투자자** | 능동적 리서치, 밸류에이션 시도, 영어 리딩 가능 | 정량 분석 도구 + 다양한 관점 |
| **C. 입문자** | 용어 어려움, 어디서 시작할지 모름 | 쉬운 해설, 교육적 분석 |

## 2. 현재 Pain Points 분석

### 2.1 정보 파편화 (가장 큰 문제)

```
현재 사용자 여정:
Yahoo Finance (재무데이터)
  → SEC EDGAR (10-K, 10-Q)
    → Google News (뉴스)
      → Seeking Alpha (분석글)
        → Excel (DCF 모델링)
          → 결론???
```

- 5~6개 사이트를 돌아다니며 수작업으로 정보 수집
- 하나의 종목 분석에 반나절~하루 소요
- 수집한 정보 간의 연결고리를 스스로 만들어야 함

### 2.2 밸류에이션의 벽

- DCF를 하고 싶지만 WACC 계산, 성장률 추정이 어려움
- 엑셀 템플릿을 구해도 어떤 가정치를 넣어야 하는지 모름
- "숫자는 넣었는데, 이 결과를 어떻게 해석해야 하지?"

### 2.3 분석 프레임워크 부재

- 뉴스와 감정에 휘둘리는 의사결정
- "버핏이라면 이 주식을 어떻게 볼까?" 라는 질문에 답할 수 없음
- 자기 확증 편향 - 사고 싶은 주식의 좋은 정보만 찾게 됨

### 2.4 영어 장벽 (한국 투자자)

- 10-K, Earnings Call Transcript가 전부 영어
- 번역기로 돌리면 재무 용어가 깨짐
- 핵심 뉘앙스(경영진의 톤, 리스크 표현)를 놓침

### 2.5 결론 도달의 어려움

- 데이터는 많은데 "그래서 사야 해? 말아야 해?"에 대한 답이 없음
- 적정주가 범위에 대한 감이 없음

## 3. 핵심 기능 설계

### 3.1 Feature 1: 원클릭 종합 분석 리포트

**해결하는 Pain Point**: 정보 파편화, 시간 부족

```
[AAPL 입력] → 클릭 한 번 →

┌─────────────────────────────────────────┐
│  Apple Inc. 종합 분석 리포트             │
│                                         │
│  재무 요약 (최근 5년 추이)               │
│  - 매출, 영업이익, FCF, 부채비율 등      │
│  - 핵심 지표 트렌드 시각화               │
│                                         │
│  산업 포지셔닝                           │
│  - 경쟁사 대비 위치                      │
│  - 산업 성장률 vs 회사 성장률            │
│                                         │
│  최근 뉴스 & 이벤트 요약                 │
│  - 긍정/부정/중립 분류                   │
│  - 주가에 영향 줄 핵심 이벤트            │
│                                         │
│  리스크 팩터                             │
│  - 재무적 리스크                         │
│  - 산업/규제 리스크                      │
└─────────────────────────────────────────┘
```

**UX 포인트**:
- 티커 입력 하나로 모든 분석이 시작됨 (진입장벽 최소화)
- 리포트 생성 중 프로그레스를 단계별로 보여줌 ("재무제표 분석 중..." → "뉴스 수집 중..." → "리포트 작성 중...")
- 리포트는 한국어로 제공, 원문 토글 가능

### 3.2 Feature 2: 전설적 투자자 관점 분석

**해결하는 Pain Point**: 분석 프레임워크 부재, 확증 편향 방지

```
┌─────────────────────────────────────────────────┐
│  AAPL - 투자 대가의 눈으로 보기                   │
│                                                   │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐          │
│  │ 버핏     │ │ 피터린치  │ │ 레이달리오│          │
│  │ 가치투자 │ │ 성장투자  │ │ 매크로   │          │
│  └──────────┘ └──────────┘ └──────────┘          │
│                                                   │
│  Warren Buffett 관점:                             │
│                                                   │
│  "경제적 해자(Moat) 평가: ★★★★☆"                  │
│  → 브랜드 파워와 생태계 전환비용이 강력한 해자     │
│  → 다만, 하드웨어 의존도가 서비스 대비 높아        │
│    마진 구조 개선 여부가 관건                      │
│                                                   │
│  "경영진 신뢰도: ★★★★★"                           │
│  → 자사주 매입에 적극적, 주주환원 정책 일관        │
│                                                   │
│  "매수 적정가: $XXX ~ $XXX"                       │
│  → 안전마진 30% 기준                              │
│                                                   │
│  종합 판단: 현재가 기준 "관심 목록 유지"           │
└─────────────────────────────────────────────────┘
```

**포함할 투자자 프레임워크**:

| 투자자 | 핵심 분석 기준 |
|--------|---------------|
| **Warren Buffett** | 경제적 해자, 경영진 품질, 안전마진, ROE |
| **Peter Lynch** | PEG 비율, 카테고리 분류(Stalwart/Fast Grower 등), 스토리 분석 |
| **Ray Dalio** | 매크로 환경, 경기 사이클 위치, 포트폴리오 내 역할 |
| **Joel Greenblatt** | Magic Formula(ROC + Earnings Yield) 순위 |
| **Benjamin Graham** | Graham Number, 재무 안전성 체크리스트 |
| **Phil Fisher** | 정성적 15포인트 체크리스트 |

**UX 포인트**:
- 각 투자자의 실제 저서/철학에 기반한 구조화된 체크리스트
- 투자자별로 이 종목에 대한 점수와 판단 근거를 명확히 제시
- "버핏은 긍정적이지만, 그레이엄 기준으로는 부채비율이 높다" 같은 관점 간 충돌을 의도적으로 보여줌 → 확증 편향 방지

### 3.3 Feature 3: 인터랙티브 밸류에이션 도구

**해결하는 Pain Point**: 밸류에이션의 벽, 결론 도달 어려움

```
┌─────────────────────────────────────────────────┐
│  AAPL 적정주가 분석                               │
│                                                   │
│  ■ DCF (Discounted Cash Flow)                     │
│    AI 추정 적정가: $198 ~ $227                    │
│    ├─ 매출 성장률:  [====|======] 8.5%            │
│    ├─ 영업이익률:   [=====|=====] 31.2%           │
│    ├─ WACC:        [===|=======] 9.2%            │
│    └─ 영구성장률:   [==|========] 2.5%            │
│                                                   │
│  ■ RIM (Residual Income Model)                    │
│    AI 추정 적정가: $205 ~ $235                    │
│                                                   │
│  ■ Comparable Analysis (멀티플 비교)               │
│    업종 평균 P/E 기준: $189                       │
│    업종 평균 EV/EBITDA 기준: $212                  │
│                                                   │
│  ■ 종합 적정주가 레인지                            │
│    ┌──────────────────────────────────┐           │
│    │  $189    $205  [$220]  $235      │           │
│    │   ◆───────●━━━━━━●───────◆      │           │
│    │         현재가: $228              │           │
│    │              ▲                   │           │
│    └──────────────────────────────────┘           │
│                                                   │
│  → 현재가는 적정가 범위 상단에 위치                 │
│  → 안전마진 확보를 위한 매수가: ~$195              │
└─────────────────────────────────────────────────┘
```

**UX 포인트**:
- AI가 먼저 합리적인 기본 가정치를 세팅해줌 (빈 엑셀의 공포 해소)
- 슬라이더로 가정치를 조정하면 실시간으로 적정주가가 변동 (인터랙티브)
- 복수의 밸류에이션 결과를 하나의 레인지 차트로 시각화
- "왜 이 가정치를 선택했는지" AI 설명 토글 가능

```
│  │ → 투자 의미: 서비스 매출 리스크 부각   │

```

### 3.5 Feature 5: 워치리스트 & 알림

**해결하는 Pain Point**: 지속적 모니터링 부담

- 관심 종목 등록 → 주요 이벤트 발생 시 알림
- "실적 발표 3일 전", "적정주가 진입", "투자 대가 기준 변화" 등 트리거
- 포트폴리오 등록 시 보유 종목 종합 리스크 대시보드

## 4. 사용자 플로우 (핵심 시나리오)

```
홈 화면
  │
  ├─ [티커 검색창] ← 가장 눈에 띄는 요소 (Google 스타일)
  │   "분석할 종목을 입력하세요 (예: AAPL, TSLA)"
  │
  ▼
분석 대시보드 (탭 구조)
  │
  ├─ [1. 종합 리포트]     ← 기본 랜딩
  ├─ [2. 투자 대가 분석]   ← 차별화 기능
  ├─ [3. 밸류에이션]      ← 정량 분석
  ├─ [4. 뉴스 & 이벤트]   ← 실시간
  └─ [5. 재무제표 원본]   ← 상세 데이터
```

## 5. 기능 우선순위 (MVP → 확장)

| 순위 | 기능 | 이유 |
|:---:|------|------|
| **P0** | 티커 검색 + 종합 리포트 | 핵심 가치. 이것만으로도 사용 이유가 됨 |
| **P0** | 투자 대가 관점 분석 | 가장 큰 차별화 포인트 |
| **P0** | 밸류에이션 도구 (DCF/RIM) | 핵심 기능, 경쟁 서비스 대비 우위 |
| **P1** | 뉴스 인텔리전스 | 리포트 품질을 높이는 보조 기능 |
| **P1** | 워치리스트 & 알림 | 리텐션을 만드는 기능 |
| **P2** | 종목 비교 도구 | "AAPL vs MSFT" 나란히 비교 |
| **P2** | 포트폴리오 분석 | 전체 보유종목 리스크/밸런스 진단 |
| **P3** | 커뮤니티/공유 | 분석 리포트 공유, 토론 |

## 6. 핵심 UX 원칙

1. **"티커 하나면 끝"** - 진입점을 극도로 단순하게
2. **"AI가 먼저, 사용자가 조정"** - 빈 화면 공포를 없애고, AI가 초안을 제시하면 사용자가 수정
3. **"판단의 근거를 항상 투명하게"** - 모든 수치와 의견에 출처/로직을 보여줌
4. **"다양한 관점을 의도적으로 충돌"** - 확증 편향 방지가 투자 분석 도구의 윤리적 책임
5. **"점진적 공개(Progressive Disclosure)"** - 요약 → 상세 → 원본 데이터 순으로 drill-down
